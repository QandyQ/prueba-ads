Última actualización: Abril 22, 2026

1. Aceptación de Términos

Al acceder y utilizar el sitio web "Lo Curioso" (en adelante, "el Sitio"), aceptas estar sujeto a estos Términos y Condiciones. Si no estás de acuerdo con alguna parte, te recomendamos que no utilices el Sitio.

2. Descripción del Servicio

Lo Curioso es una plataforma educativa dedicada a proporcionar contenido informativo sobre procesos de fabricación y producción de diversos productos. El contenido está disponible de forma gratuita y se proporciona solo con fines educativos e informativos.

3. Uso Permitido

Aceptas utilizar el Sitio únicamente para propósitos legales y de manera consistente con estos Términos. Específicamente, aceptas no:

  • Usar el Sitio para cualquier propósito ilegal o no autorizado
  • Interferir con la funcionalidad, seguridad o acceso del Sitio
  • Reproducir, distribuir o transmitir contenido sin autorización expresa
  • Realizar ingeniería inversa o intentar acceder a código fuente
  • Usar bots, scrapers u herramientas automatizadas
  • Realizar ataques de denegación de servicio (DDoS)
  • Acosar, amenazar o insultar a otros usuarios o al equipo de Lo Curioso

4. Contenido del Usuario

Si envías comentarios, sugerencias o cualquier contenido a través de nuestro formulario de contacto o redes sociales:

  • Nos autorizas a usar ese contenido (atribuyéndote si lo publicamos)
  • Garantizas que el contenido es tuyo o tienes permiso para compartirlo
  • El contenido no viola derechos de terceros

No se permite: Reproducir, modificar o usar comercialmente el contenido sin permiso.

6. Enlaces Externos

El Sitio puede contener enlaces a sitios web de terceros. No somos responsables del contenido, precisión o prácticas de privacidad de esos sitios. Úsalos bajo tu propio riesgo.
